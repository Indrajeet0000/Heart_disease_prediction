The Heart Disease Prediction System uses various health-related features such as age, blood pressure, cholesterol level, heart rate, and more to predict whether a person is at risk of heart disease.

The model is trained on a medical dataset and provides accurate predictions using classification algorithms. The project also includes a simple user interface where users can input their details and get instant results.

🚀 Features
Predicts heart disease risk instantly
User-friendly interface
Data preprocessing and normalization
Multiple feature inputs (Age, BP, Cholesterol, etc.)
Trained Machine Learning model
Real-time prediction output
🛠️ Technologies Used
Python
Machine Learning (Scikit-learn)
Pandas & NumPy
Matplotlib & Seaborn
Flask / Streamlit (for UI)
Jupyter Notebook
📊 Input Features
Age
Resting Blood Pressure
Cholesterol
Maximum Heart Rate
Oldpeak
Chest Pain Type
Fasting Blood Sugar
Resting ECG
Exercise Induced Angina
🤖 Machine Learning Model
Algorithm Used: K-Nearest Neighbors (KNN) / Logistic Regression
Data Scaling using StandardScaler
Model Evaluation using Accuracy Score
📈 Output
0 → No Heart Disease
1 → High Risk of Heart Disease
